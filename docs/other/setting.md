1 git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
2 通过在~/.zshrc中加入插件的名字 plugins=( [plugins...] zsh-syntax-highlighting)
3 执行 source ~/.zshrc
